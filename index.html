HTML 기본 태그 익히기

웹사이트를 만들기 위한 첫 걸음! HTML 태그들을 하나씩 배워보자.

📚 학습 목표

  • HTML 문서의 기본 구조 이해하기
  • 제목 태그 (h1~h6) 활용법 익히기
  • 문단과 목록 태그 사용하기
  • 이미지와 링크 태그 다루기
  • 시맨틱 마크업의 중요성 알기

🏗️ HTML 문서 기본 구조

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>페이지 제목</title>
</head>
<body>
    <!-- 페이지 내용 -->
</body>
</html>

DOCTYPE: HTML5 문서임을 선언

html: HTML 문서의 루트 요소

head: 메타데이터 영역 (브라우저에 표시되지 않음)

body: 실제 콘텐츠 영역 (브라우저에 표시됨)

📝 제목과 문단 태그

<h1>~<h6>
제목 태그 (가장 중요한 h1부터 h6까지)

Main Title (h1)

Subtitle (h2)

Section Title (h3)

<p>
문단 태그 (paragraph)

이것은 문단 태그입니다. 긴 텍스트를 구조적으로 나누어 표현할 때 사용합니다.

📋 목록 태그

순서 없는 목록 (ul)

<ul>
    <li>첫 번째 항목</li>
    <li>두 번째 항목</li>
    <li>세 번째 항목</li>
</ul>
  • 웹 디자인 기초
  • HTML/CSS 마스터
  • 자바스크립트 활용

순서 있는 목록 (ol)

<ol>
                    <li>HTML 구조 익히기</li>
                    <li>CSS 스타일링</li>
                    <li>JavaScript 로직</li>
</ol>
  1. HTML 구조 익히기
  2. CSS 스타일링
  3. JavaScript 로직

🖼️ 이미지와 링크

이미지 태그 (img)

<img src="이미지경로.jpg" alt="이미지 설명" width="300">

링크 태그 (a)

<a href="https://www.w3schools.com" target="_blank">
    W3Schools에서 더 배우기
</a>
W3Schools HTML 튜토리얼 →

📖 정의 목록 (dl, dt, dd)

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language의 줄임말</dd>
    
    <dt>CSS</dt>
    <dd>Cascading Style Sheets의 줄임말</dd>
</dl>
HTML
웹 페이지의 구조를 정의하는 마크업 언어
CSS
웹 페이지의 스타일과 레이아웃을 담당하는 스타일시트 언어
JavaScript
웹 페이지에 동적 기능을 추가하는 프로그래밍 언어

💻 실습 예제

개인 소개 페이지 만들기

배운 태그들을 활용하여 간단한 자기소개 페이지를 만들어보세요:

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>나를 소개합니다</title>
</head>
<body>
    <h1>안녕하세요! 저는 [이름]입니다 😊</h1>
    
    <p>웹 개발을 배우고 있는 신입 개발자입니다.</p>
    
    <h2>나의 취미</h2>
    <ul>
        <li>코딩하기</li>
        <li>새로운 기술 탐구</li>
        <li>온라인 강의 수강</li>
    </ul>
    
    <h2>학습 계획</h2>
    <ol>
        <li>HTML/CSS 마스터</li>
        <li>JavaScript 기초</li>
        <li>React 프레임워크</li>
    </ol>
</body>
</html>

💡 개발자 팁

🎯 시맨틱 마크업

태그를 의미에 맞게 사용하세요. h1은 페이지당 하나만, 제목은 순서대로 사용하는 것이 좋습니다.

♿ 접근성

img 태그에는 항상 alt 속성을, 링크에는 의미있는 텍스트를 사용하세요.

📱 반응형

viewport 메타태그를 꼭 추가하여 모바일에서도 잘 보이도록 하세요.

🔧 개발자 도구

F12키로 개발자 도구를 열어 HTML 구조를 실시간으로 확인해보세요.
